## 📥 Getting Started

### Prerequisites
- Android Studio Hedgehog+
- JDK 17
- Android SDK 24 (Nougat) or higher

### Installation Steps
1. **Clone the Repo**
   ```bash
   git clone https://github.com/YourUsername/MOD-STORE-APP.git
   ```
2. **Configure Secrets**
   Add your GitHub token to `local.properties`:
   ```properties
   github.token=ghp_example123
   ```
3. **Compile**
   Press `Shift + F10` in Android Studio to build the **Release APK**.
